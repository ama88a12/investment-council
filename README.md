# 🏛️ مجلس الاستثمار السري

تطبيق تفاعلي يجمع عمالقة الاستثمار في "غرفة قرار" واحدة لتحليل الفرص المالية والجيوسياسية.

## 🌟 المستثمرون المشاركون
* **وارن بافيت**: تحليل القيمة والخندق الاقتصادي.
* **تشارلي مونغر**: النماذج العقلانية وكشف الأخطاء.
* **راي داليو**: دورات الديون والنظم الكلية.
* **جورج سوروس**: الانعكاسية وفجوات السوق.
* **بيتر لينش**: النمو العملي والفرص البسيطة.
* **كاثي وود**: الابتكار التكنولوجي المستقبلي.

## 🛠️ التقنيات
* React.js
* Tailwind CSS
